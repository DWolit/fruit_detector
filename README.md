# Подсчет фруктов на изображении

Веб-приложение для автоматического обнаружения и подсчёта фруктов на фотографии с помощью нейронной сети YOLOv8.

## Что умеет

- Загрузить фотографию с фруктами
- Автоматически найти яблоки, бананы и апельсины
- Нарисовать цветные рамки вокруг каждого фрукта с процентом уверенности
- Исправить ошибки вручную (сменить метку или удалить лишнее)
- Сохранить результат в историю
- Скачать историю в Excel

## Требования

- Python 3.8 и выше
- Интернет при первом запуске (скачается модель ~6 МБ)

## Запуск

**Linux / Mac:**
```bash
bash run.sh
```

**Windows:**
```
run.bat
```

После запуска открыть в браузере: **http://localhost:5000**

## Стек

| Технология | Для чего |
|---|---|
| Python + Flask | веб-сервер и API |
| YOLOv8 (Ultralytics) | нейронная сеть для детекции |
| OpenCV | чтение изображений |
| SQLite | хранение истории |
| openpyxl | экспорт в Excel |
| HTML5 Canvas + JS | отображение рамок в браузере |

## Структура проекта

```
project/
├── app.py               — серверная часть (Flask)
├── requirements.txt     — зависимости
├── run.sh               — запуск Linux/Mac
├── run.bat              — запуск Windows
└── templates/
    ├── index.html       — главная страница
    └── history.html     — история результатов
```

После первого запуска появятся:
- `yolov8n.pt` — веса модели (скачиваются автоматически)
- `history.db` — база данных SQLite
